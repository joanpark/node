Welcome to Opus

a minimal note-taking app.

Open a folder
Opus opens a window for each folder. Notes are stored as regular files, so you can keep them in any folder.
Create a note
Create new notes by right-clicking folders in the sidebar. The sidebar only shows files with the .note extension to avoid clutter.
Write!
The menu contains formatting options and other features. For more information, read the documentation.
